# Backup Kit

Use this folder as the pitch-day fallback package.

## Purpose

- Keep the strongest demo artifacts in one place.
- Avoid relying on a live synthesis run if the laptop is slow.
- Give presenters a static path: screenshot, video, ZIP export, report JSON, and latest pitch summary.

## Expected Contents

- `demo_runs.json`: manifest powering the saved demo-run cards in the app
- `primary-distributed-network.zip`: recommended run export package
- `backup-ed-surge.zip`: backup run export package
- `primary-report.json`: report for the recommended run
- `backup-report.json`: report for the backup run
- `latest-summary.md`: latest pitch-summary markdown copied from the best run
